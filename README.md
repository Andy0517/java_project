# java_project
終極密碼

介紹
不管你是自己1人，還是朋友聚餐，
都十分適合的小遊戲
單人-你是猜數字高手嗎?趕快來猜猜
，創造得分的巔峰
雙人-適合寓教於樂，老少咸宜
規則
單人-每輪得100分，輸入錯誤扣200
分，每5輪扣300分，如果幸運
數字和終極密碼一樣得500分
雙人-輸入雙方姓名並自訂最大數，
便可以開始遊戲，猜到的人就
輸了
